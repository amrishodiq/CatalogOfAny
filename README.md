CatalogOfAny
============

Catalog of Any is a multipurpose note taking Android app. Technically, what this app do is CRUD to SQLite database. 

However, since this app uses images taken from local storage and display them in card/list style, an image caching method implemented. To see this app, please download it at Google Play


Download
========
Location: https://play.google.com/store/apps/details?id=me.shodiq.amri.catalogofany
Requirement: Android 4.2
